## Requisitos

- Node.js 18.18 (Latest em 10-05-2023)
- Docker
- Docker-compose

## Instalação

```bash 
# 1. Clone o projeto
git clone https://github.com/fga-eps-mds/2024.1-PrintGo-FrontEnd.git

# 2. Entre na pasta do projeto
cd 2024.1-PrintGo-FrontEnd
# 3. Rodar o docker
docker-compose up --build
```
